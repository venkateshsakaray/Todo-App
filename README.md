# Todo App 📝

A fully functional Todo App built with React.js

## Live Demo
🔗 https://todo-app-phi-blond.vercel.app

## Features
- Load todos automatically from REST API
- Add new todos
- Mark todos as complete/incomplete
- Delete todos
- Professional dark theme UI
- Responsive design

## Tech Stack
- React.js
- JavaScript (ES6+)
- CSS3
- Vite
- REST API (JSONPlaceholder)
- Deployed on Vercel

- useState and useEffect hooks
- Fetching data from REST API
- Props and component breakdown
